# Hello World

Welcome to our modern web application. This is our first blog post written in MDX format.

## What's MDX?

MDX is a powerful format that lets you seamlessly write JSX in your Markdown documents.
You can import components, such as interactive charts or alerts, and embed them within
your content.

## Features

- Write markdown with JSX
- Import and use React components
- Create interactive blog posts
- Style with Tailwind CSS
