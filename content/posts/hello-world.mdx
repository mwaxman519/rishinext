# Hello World

Welcome to our modern web application. This is our first blog post written in MDX format.

## What's MDX?

MDX is a powerful format that lets you seamlessly write JSX in your Markdown documents.
You can import components, such as interactive charts or alerts, and embed them within
your content.

<Callout type="info">
  MDX allows you to use React components directly in your markdown content!
</Callout>

## Code Example

```typescript
// Example of TypeScript code with syntax highlighting
interface User {
  id: string;
  name: string;
  email: string;
}

function greetUser(user: User) {
  return `Hello, ${user.name}!`;
}
```
